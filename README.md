## Features

- Step-by-step drills covering modes, movement, text objects, visual mode, search/replace, and spf13 shortcuts
- Shows correct keystrokes in **green**, mistakes in **red**
- Wrong key → instant skip to next exercise
- Keeps full history visible (no screen clearing between drills)
- Final score with accuracy % and average time per correct drill
